04. statistical inference · t-test & ci

frequentist basics: t-test, confidence intervals, p-values.

python · scipy.stats
import numpy as np
from scipy import stats

np.random.seed(42)
sample_a = np.random.normal(loc=100, scale=10, size=50)
sample_b = np.random.normal(loc=105, scale=12, size=45)

t_stat, p_value = stats.ttest_ind(sample_a, sample_b)
print(f"t-test: t = {t_stat:.3f}, p = {p_value:.4f}")

diff = sample_b.mean() - sample_a.mean()
se = np.sqrt(sample_a.var()/len(sample_a) + sample_b.var()/len(sample_b))
ci = (diff - 1.96*se, diff + 1.96*se)
print(f"mean difference (B-A): {diff:.2f}, 95% CI {ci}")

05. SQL for data wrangling (sqlite3)

run sql directly on dataframes — essential data skill.

python · sqlite3
import sqlite3
import pandas as pd

df = pd.DataFrame({
    'user_id': [1,2,3,1,4,2],
    'action': ['click', 'view', 'click', 'purchase', 'view', 'click'],
    'value': [1.2, 0.5, 1.2, 35.0, 0.5, 1.2]
})

conn = sqlite3.connect(':memory:')
df.to_sql('events', conn, index=False, if_exists='replace')

query = """
SELECT user_id, 
       COUNT(*) as num_actions,
       SUM(value) as total_value
FROM events
GROUP BY user_id
ORDER BY total_value DESC;
"""
result = pd.read_sql(query, conn)
print("user summary:")
print(result)
conn.close()

06. applied machine learning (sklearn)

random forest classifier, train/test split, metrics.

python · scikit-learn
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.metrics import classification_report

X, y = make_classification(n_samples=500, n_features=7, n_informative=5, random_state=1)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, stratify=y)

rf = RandomForestClassifier(n_estimators=50, max_depth=5)
rf.fit(X_train, y_train)
pred = rf.predict(X_test)

print("classification report:")
print(classification_report(y_test, pred, target_names=['class0','class1']))
print("feature importances:", rf.feature_importances_.round(3))

07. big data basics (pyspark style)

simulated spark aggregations using pandas — identical semantics.

python · spark-like
import pandas as pd

df = pd.DataFrame({
    'department': ['sales', 'eng', 'sales', 'hr', 'eng'],
    'salary': [85000, 110000, 92000, 62000, 115000],
    'experience': [3, 7, 4, 2, 8]
})

result = df.groupby('department').agg(
    avg_salary=('salary', 'mean'),
    max_exp=('experience', 'max')
).reset_index()

print("department summary (mock spark):")
print(result.sort_values('avg_salary', ascending=False))

08. ethics · deployment · communication

bias checks, model cards, and simple fairness metrics.

python · fairness check
import numpy as np

np.random.seed(1)
pred_male = np.random.binomial(1, 0.7, size=100)
pred_female = np.random.binomial(1, 0.6, size=80)

print("positive rate male: ", pred_male.mean())
print("positive rate female:", pred_female.mean())
diff_ratio = pred_male.mean() / pred_female.mean()
print(f"disparate impact (should be >0.8): {diff_ratio:.2f}")
