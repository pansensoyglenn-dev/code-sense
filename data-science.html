02 — PYTHON DATA STACK

pandas & NumPy

pandas for dataframes, numpy for vectorized ops — the python blue duo.

python · pandas/numpy
import pandas as pd
import numpy as np

df = pd.DataFrame({
    'product': ['A', 'B', 'A', 'C', 'B'],
    'qty': [5, 12, 8, 3, 15],
    'price': [120, 85, 120, 200, 85]
})
df['revenue'] = df['qty'] * df['price']
pivot = df.groupby('product')['revenue'].sum().reset_index()
print("revenue by product:")
print(pivot)
print("\ndescriptive stats:")
print(df.describe())

03 — VISUALIZATION & EDA

seaborn & matplotlib

exploratory plots: distributions, correlations, outliers.

python · seaborn/matplotlib
import matplotlib.pyplot as plt
import seaborn as sns
import pandas as pd

df = sns.load_dataset('penguins').dropna()

print("avg bill length by species:")
print(df.groupby('species')['bill_length_mm'].mean())

plt.figure(figsize=(6,4))
sns.histplot(data=df, x='flipper_length_mm', hue='species', alpha=0.5)
plt.title('flipper length distribution')

print("\nflipper length range:", df['flipper_length_mm'].min(), '-', df['flipper_length_mm'].max())
print("bill–flipper correlation:", df['bill_length_mm'].corr(df['flipper_length_mm']).round(3))

04 — STATISTICAL INFERENCE

t-test & confidence intervals

frequentist basics: t-test, confidence intervals, p-values.

python · scipy.stats
import numpy as np
from scipy import stats

np.random.seed(42)
sample_a = np.random.normal(loc=100, scale=10, size=50)
sample_b = np.random.normal(loc=105, scale=12, size=45)

t_stat, p_value = stats.ttest_ind(sample_a, sample_b)
print(f"t-test: t = {t_stat:.3f}, p = {p_value:.4f}")

diff = sample_b.mean() - sample_a.mean()
se = np.sqrt(sample_a.var()/len(sample_a) + sample_b.var()/len(sample_b))
ci = (diff - 1.96*se, diff + 1.96*se)
print(f"mean difference (B-A): {diff:.2f}, 95% CI {ci}")

05 — SQL FOR DATA WRANGLING

sqlite3 in Python

run sql directly on dataframes — essential data skill.

python · sqlite3
import sqlite3
import pandas as pd

df = pd.DataFrame({
    'user_id': [1,2,3,1,4,2],
    'action': ['click', 'view', 'click', 'purchase', 'view', 'click'],
    'value': [1.2, 0.5, 1.2, 35.0, 0.5, 1.2]
})

conn = sqlite3.connect(':memory:')
df.to_sql('events', conn, index=False, if_exists='replace')

query = """
SELECT user_id, 
       COUNT(*) as num_actions,
       SUM(value) as total_value
FROM events
GROUP BY user_id
ORDER BY total_value DESC;
"""
result = pd.read_sql(query, conn)
print("user summary:")
print(result)
conn.close()

06 — APPLIED MACHINE LEARNING

scikit-learn classifier

random forest classifier, train/test split, metrics.

python · scikit-learn
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import make_classification
from sklearn.model_selection import train_test_split
from sklearn.metrics import classification_report

X, y = make_classification(n_samples=500, n_features=7, n_informative=5, random_state=1)
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, stratify=y)

rf = RandomForestClassifier(n_estimators=50, max_depth=5)
rf.fit(X_train, y_train)
pred = rf.predict(X_test)

print("classification report:")
print(classification_report(y_test, pred, target_names=['class0','class1']))
print("feature importances:", rf.feature_importances_.round(3))

07 — BIG DATA BASICS

pyspark style aggregations

simulated spark aggregations using pandas — identical semantics.

python · spark-like
import pandas as pd

df = pd.DataFrame({
    'department': ['sales', 'eng', 'sales', 'hr', 'eng'],
    'salary': [85000, 110000, 92000, 62000, 115000],
    'experience': [3, 7, 4, 2, 8]
})

result = df.groupby('department').agg(
    avg_salary=('salary', 'mean'),
    max_exp=('experience', 'max')
).reset_index()

print("department summary (mock spark):")
print(result.sort_values('avg_salary', ascending=False))

08 — ETHICS & DEPLOYMENT

fairness & model cards

bias checks, model cards, and simple fairness metrics.

python · fairness check
import numpy as np

np.random.seed(1)
pred_male = np.random.binomial(1, 0.7, size=100)
pred_female = np.random.binomial(1, 0.6, size=80)

print("positive rate male: ", pred_male.mean())
print("positive rate female:", pred_female.mean())
diff_ratio = pred_male.mean() / pred_female.mean()
print(f"disparate impact (should be >0.8): {diff_ratio:.2f}")

print("\nmodel card snippet: random forest, acc 0.91, trained 2023.")

🧠 responsible ds: always audit for bias, document with model cards, maintain versioning.

FAQ

Frequently Asked Questions

The core Python data science stack includes: NumPy for numerical computing, pandas for data manipulation, Matplotlib/Seaborn for visualization, Scikit-learn for machine learning, and SciPy for scientific computing. For big data, PySpark is commonly used.

pandas provides several methods: dropna() removes missing values, fillna() replaces them with a specified value or method (ffill, bfill), and interpolate() fills by interpolation. Use isnull() to detect missing values and notnull() for non-missing.

In scikit-learn, fit() learns parameters from training data, transform() applies the transformation to data, and fit_transform() does both in one step for efficiency. Use fit() on training data, then transform() on both train and test sets.
