10 Core Programming Concepts for the Quantum Era

01. Qubit Initialization

Unlike a classical bit (0 or 1), a Qubit starts in a ground state but can hold complex probabilities.

qreg q[1]; // Declare 1 quantum register creg c[1]; // Declare 1 classical register

02. Hadamard Gate (Superposition)

The H-Gate puts a qubit into a state where it is 50% zero and 50% one simultaneously.

h q[0]; // Apply Hadamard gate // State: (|0> + |1>) / √2

03. Pauli-X Gate (Quantum NOT)

The X-Gate flips the state of a qubit, acting as the quantum equivalent of the classical NOT gate.

x q[0]; // Flips |0> to |1> or vice versa

04. CNOT Gate (Entanglement)

Controlled-NOT links two qubits. If the control qubit is 1, the target qubit flips. This creates entanglement.

cx q[0], q[1]; // Control q0, Target q1

05. Measurement

Measurement forces the quantum "wavefunction" to collapse into a single classical 0 or 1 value.

measure q[0] -> c[0]; // Collapse to classical bit

06. Bell State Pair

Combining Hadamard and CNOT creates the simplest form of quantum entanglement between two particles.

h q[0]; cx q[0], q[1]; // Result: Perfectly correlated qubits

07. Quantum Teleportation

Logic to move the state of one qubit to another using a pre-entangled pair and classical communication.

// Transferring state from q0 to q2 h q[1]; cx q[1], q[2]; cx q[0], q[1]; h q[0];

08. Grover’s Algorithm (Search)

Quantum logic that finds an item in an unsorted database of $N$ items in roughly $\sqrt{N}$ steps.

// Oracle implementation h q[0..n]; oracle(q); diffusion(q);

09. Quantum Key Distribution (BB84)

Uses quantum mechanics to ensure secure communication. Any eavesdropping attempt collapses the state.

if (basis_match) { keep_key_bit(); } else { discard(); }

10. Toffoli Gate (CCNOT)

A universal reversible logic gate with two control bits and one target. Essential for quantum arithmetic.

ccx q[0], q[1], q[2]; // If q0 & q1 are 1, flip q2
