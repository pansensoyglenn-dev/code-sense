02 — LLM & TRANSFORMERS

Hugging Face Pipelines

Use state-of-the-art language models via the transformers library. Below: text generation with GPT-2, sentiment analysis, and zero-shot classification.

# Text generation with GPT-2 from transformers import pipeline generator = pipeline('text-generation', model='gpt2') prompt = "Once upon a time in an AI lab," output = generator(prompt, max_length=50, num_return_sequences=1) print(output[0]['generated_text']) # Sentiment analysis classifier = pipeline('sentiment-analysis') result = classifier("I love building AI programs with Python!") print(result) # [{'label': 'POSITIVE', 'score': 0.999...}] # Zero-shot classification classifier_zs = pipeline("zero-shot-classification") candidate_labels = ["technology", "sports", "finance"] text = "NVIDIA just launched a new GPU for deep learning." print(classifier_zs(text, candidate_labels))

03 — NLP PIPELINES

NER, Question Answering, Summarization

Perform named entity recognition (NER), extractive QA, and text summarization with minimal code using Hugging Face pipelines.

# Named Entity Recognition ner = pipeline("ner", grouped_entities=True) text = "Elon Musk founded SpaceX in Hawthorne, California." entities = ner(text) for ent in entities: print(f"{ent['word']} -> {ent['entity_group']}") # Question Answering qa = pipeline("question-answering") context = "The Python programming language was created by Guido van Rossum and released in 1991." question = "Who created Python?" answer = qa(question=question, context=context) print(f"Answer: {answer['answer']} (confidence: {answer['score']:.2f})") # Summarization summarizer = pipeline("summarization", model="facebook/bart-large-cnn") long_text = "Artificial intelligence (AI) refers to the simulation of human intelligence in machines that are programmed to think like humans and mimic their actions. The term may also be applied to any machine that exhibits traits associated with a human mind such as learning and problem-solving." summary = summarizer(long_text, max_length=50, min_length=25) print(summary[0]['summary_text'])

04 — COMPUTER VISION

YOLO Object Detection

Real-time object detection using YOLOv8. The example shows inference on an image; requires ultralytics and PIL.

from ultralytics import YOLO from PIL import Image import requests # Load a pretrained YOLOv8 model model = YOLO('yolov8n.pt') # nano version for speed # Run inference on an image from URL image_url = 'https://ultralytics.com/images/bus.jpg' img = Image.open(requests.get(image_url, stream=True).raw) results = model(img) # returns list of Results # View detections for r in results: for box in r.boxes: cls = int(box.cls[0]) conf = float(box.conf[0]) print(f"Class: {model.names[cls]}, Confidence: {conf:.2f}")

05 — SPEECH-TO-TEXT

Whisper by OpenAI

Transcribe audio files using OpenAI's Whisper model via Hugging Face.

import torch from transformers import AutoModelForSpeechSeq2Seq, AutoProcessor, pipeline device = "cuda:0" if torch.cuda.is_available() else "cpu" model_id = "openai/whisper-small" model = AutoModelForSpeechSeq2Seq.from_pretrained(model_id).to(device) processor = AutoProcessor.from_pretrained(model_id) pipe = pipeline( "automatic-speech-recognition", model=model, tokenizer=processor.tokenizer, feature_extractor=processor.feature_extractor, device=device, ) result = pipe("path/to/audio.wav") # replace with your audio file print(result["text"])

06 — GENERATIVE AI

Simple LLM Chatbot

A minimal example demonstrating a chat loop with a language model using DialoGPT.

from transformers import pipeline, Conversation chatbot = pipeline("conversational", model="microsoft/DialoGPT-medium") conversation = Conversation("What is the future of AI?") conversation = chatbot(conversation) print(conversation.generated_responses[-1]) # Continue conversation conversation.add_user_input("Will AI replace programmers?") conversation = chatbot(conversation) print(conversation.generated_responses[-1])

07 — CLASSIC ML

Sentiment Analysis with Scikit-learn

Bag-of-words + logistic regression for sentiment classification (toy example).

from sklearn.feature_extraction.text import CountVectorizer from sklearn.linear_model import LogisticRegression from sklearn.pipeline import make_pipeline # Tiny dataset texts = ["I love this!", "This is terrible", "Amazing product", "Worst purchase ever"] labels = [1, 0, 1, 0] # 1 positive, 0 negative model = make_pipeline(CountVectorizer(), LogisticRegression()) model.fit(texts, labels) print(model.predict(["Fantastic quality"])) # [1] print(model.predict(["Very disappointing"])) # [0]

08 — RAG PATTERN

Retrieval-Augmented Generation Sketch

Conceptual example of RAG: retrieve relevant context from documents, then feed to LLM. Uses sentence-transformers and transformers.

from sentence_transformers import SentenceTransformer, util from transformers import pipeline # 1. Embedding documents docs = [ "Python is a high-level programming language.", "AI programs often use PyTorch or TensorFlow.", "Transformers revolutionized NLP in 2017." ] encoder = SentenceTransformer('all-MiniLM-L6-v2') doc_emb = encoder.encode(docs, convert_to_tensor=True) # 2. Retrieve relevant doc for a query query = "What framework is used for deep learning?" query_emb = encoder.encode(query, convert_to_tensor=True) scores = util.cos_sim(query_emb, doc_emb)[0] best_idx = scores.argmax().item() context = docs[best_idx] # 3. Generate answer with LLM (mock) generator = pipeline('text-generation', model='gpt2') prompt = f"Context: {context}\nQuestion: {query}\nAnswer:" print(generator(prompt, max_length=50)[0]['generated_text'])

09 — PROMPT ENGINEERING

Structured Output Example

Demonstrates a simple prompt engineering technique for structured output (JSON).

import json from transformers import pipeline generator = pipeline('text-generation', model='gpt2') prompt = """ You are an AI assistant that outputs valid JSON. User: Extract the name, age, and city from: "John is 25 and lives in New York" AI: { "name": "John", "age": 25, "city": "New York" } User: Extract from: "Maria, age 32, from Barcelona" AI: """ output = generator(prompt, max_length=150, temperature=0.3)[0]['generated_text'] print(output)

FAQ

Frequently Asked Questions

Transformers use self-attention mechanisms to process all tokens in parallel, capturing long-range dependencies better than RNNs/LSTMs. They enable transfer learning via pre-trained models like BERT and GPT, achieving state-of-the-art results on most NLP tasks.

Install the ultralytics package and load a pretrained model: from ultralytics import YOLO; model = YOLO('yolov8n.pt'); results = model('image.jpg'). The results contain bounding boxes, class labels, and confidence scores.

Retrieval-Augmented Generation (RAG) combines a retrieval system (to fetch relevant documents) with a generative LLM. It improves factual accuracy by grounding responses in external knowledge, reducing hallucinations.
